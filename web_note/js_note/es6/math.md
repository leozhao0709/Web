# Math

## 1. Math.sign

Math.sign will give the 1/-1/0 for indicate a positive/negative/0 of the number.

```js
// 1
let number = 10;
console.log(Math.sign(number));

// -1
number = -10;
console.log(Math.sign(number));

// 0
number = 0;
console.log(Math.sign(number));
```

## 2. Math.trunc

Math.trunc will cut the part after the dot.

```js
// 3
number = 3.78;
console.log(Math.trunc(number));

// -3
number = -3.78;
console.log(Math.trunc(number));
```
